## Что это

Один HTML-файл, который превращает «список всего про AI» в навигируемую карту: что учить, в каком порядке и **что именно нужно к твоему грейду**. Без воды, без хайпа, с упором на applied-инженерию (интеграция готовых моделей, RAG, агенты, eval, прод), а не на ресёрч и обучение моделей.

Контент отражает практику mid-2026: актуальные API, инструменты и подходы, сверенные с roadmap.sh, Hugging Face Learn, Google AI и OpenAI Cookbook.

## Возможности

- **Живая карта компетенций.** Слева — 6 столпов (Python/бэкенд, LLM+промпт, RAG, агенты, eval, прод). При прокрутке шагов справа карта подсвечивает релевантный столп и показывает строку «зачем это в общей картине». Наведение/клик по столпу подсвечивает и проматывает связанные шаги.
- **Фильтр по грейдам — Junior / Middle / Senior.** Кумулятивно (Middle = Junior+Middle, Senior = всё). Нерелевантные темы мягко приглушаются; у каждого инструмента — цветная точка минимального грейда (J/M/S).
- **Drill-down по каждому инструменту.** Клик по чипу открывает панель «Что нужно знать» + «Ресурсы, где пощупать»: официальные доки, GitHub-репозитории, бесплатные курсы и статьи. **Никаких платных курсов и книг.**
- **Трекинг прогресса.** Отмечай шаги «освоено», следи за прогресс-кольцом, кнопка «Продолжить» ведёт к следующему незакрытому шагу.
- **Шеринг через ссылку.** Прогресс и выбранный грейд кодируются в URL (`#done=…&grade=…`) — отправил ссылку, у получателя восстановилось состояние.
- **Тёмная тема, мобайл, доступность.** Клавиатурная навигация, focus-trap в панелях, корректные ARIA-роли, адаптив до телефона.

## Как пользоваться

1. Открой [сайт](https://gryzb.github.io/ai-engineer-roadmap/).
2. Выбери грейд вверху правой колонки — увидишь, какие темы нужны именно тебе.
3. Кликай по чипам, чтобы провалиться в тему и получить ссылки на бесплатные ресурсы.
4. Отмечай освоенное (кружок справа от шага), следи за прогрессом.
5. Жми «Поделиться», чтобы сохранить/отправить прогресс ссылкой.

## Технологии

- Чистый **HTML + CSS + ванильный JS** в одном файле.
- Внешняя зависимость только одна — шрифты Google Fonts с CDN.
- **Без сборки, без фреймворков, без бэкенда.** Открывается двойным кликом.
- Прогресс хранится в URL (без `localStorage`), поэтому состояние легко шарить ссылкой.

## Запуск локально

```bash
git clone https://github.com/gryzb/ai-engineer-roadmap.git
cd ai-engineer-roadmap
# просто открой index.html в браузере, либо подними локальный сервер:
python3 -m http.server 8000
# → http://localhost:8000
```

## Дисклеймер

- Контент отражает практику и инструменты на **середину 2026 года**; экосистема меняется быстро — проверяйте версии и ссылки.
- Разметка тем по грейдам — авторский ориентир, а не отраслевой стандарт.
- Проект **не аффилирован** с Anthropic, OpenAI, Google, Hugging Face и другими упомянутыми компаниями. Все ссылки ведут на их официальные публичные ресурсы.
